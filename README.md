# Marketing website UI kit

React (Babel) recreation of the Inmuebles Audiovisuales marketing site.

## Files
- `index.html` — interactive demo with nav, hero, services, portfolio tabs, pricing, CTA, footer
- `Header.jsx`, `Footer.jsx`, `Hero.jsx`, `StatsBar.jsx`, `TrustBar.jsx`, `Portfolio.jsx`, `ServicesGrid.jsx`, `PricingTable.jsx`, `ProcessSteps.jsx`, `CTASection.jsx`, `Buttons.jsx`, `Icons.jsx`

All components read tokens from `../../colors_and_type.css`.
